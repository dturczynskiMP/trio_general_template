Ogólny szablon projektu dla kontrolerów ruchu TRIO Motion
=
Zawiera przykładową implementację maszyny stanów:
- DISABLED
- ACTIVE
- HOMING
- AUTOMATIC
- MANUAL
- JOG
- ERROR

Szablon jest przygotowany do współpracy z panelem HMI poprzez protokół Modbus TCP/IP.

W tym konkretnym przypadku, projekt jest przygotowany pod HMI marki Weintek serii cMT oraz iE /iP, ale powinien współpracować z każdym innym kleintem (masterem) Modbus TCP/IP

Projekt TRIO zostały stworzone w Motion Perfect 5.6.3 
Projekty Weintek zostały stworzone w EasyBuilder Pro 6.10.02.300


Aby pobrać projekt do siebie można użyć przycisku Code -> Clone lub Clone -> Download ZIP

General TRIO Project Template
=
Includes a sample implementation of a state machine:
- DISABLED
- ACTIVE
- HOMING
- AUTOMATIC
- MANUAL
- JOG
- ERROR

The template is designed to communicate with an HMI panel via the Modbus TCP/IP protocol.

In this case, the project is configured for Weintek cMT and iE / iP series HMI panels, however it should work with any Modbus TCP/IP client (master).

The TRIO project was developed in Motion Perfect 5.6.3.
The Weintek project was developed in EasyBuilder Pro 6.10.02.300.

To download the project, use the Code -> Clone or Clone -> Download ZIP button
